PrepCompanion

\-------------------



I want to build a project which aims at help people in their preparation Journey.

It helps them in tracking their progress throughout their preparation journey by suggesting Projects to be build, Certifications to take, Technical events to attend(offline/online), Daily activity tracker and Progress manager.



In this Application, I mainly want to include 4 components:

1. Project Monitor
2. Certification Tracker
3. Webinar Tracker
4. Daily Reporter

Project Monitor

\------------------

Based on the skillset and the target job role, I'm going to suggest the following Projects from Beginner to Advanced Level

Save projects that interests you by clicking on "ADD" button

Then it will add those projects to "Saved".

If we click on the project, It will ask you whether you need any assistance in beginning the project? and offer help like

* Suggesting tech stack to be learnt to build the project
* How to start
* what should be included in the project

Based on the help they took, it then ask them how much time will they took to learn the required skill and start with the project

Once they given that date/duration, it starts tracking them what they learnt and build everyday.
Also, it ask them the date from they start building the project.
Based on the progress the user is updating, it will then notify and suggest them the next relevant steps. Until they complete the project.





Certification Tracker

\------------------------

* Based on the Job role the user is preparing for and the tech stack, it will start suggesting interesting and relevant certifications that may add weight to their resume.
* Also, the user can manually add some certification to the tracker, so that it will analyze the certification details like the start date and the due date and track them.
* Notify the user to take relevant action based on the progress.
* Also, the user can update daily progress for each certification he did. So that, it can help them by giving the daily targets.
* Also, once the certification is completed, it will celebrate that by posting it in LinkedIn and other career portals.





Webinar Tracker

\--------------------

* This will suggest the relevant technical events/webinar(online/offline) that are happening around based on the chosen tech stack. 
* It will suggest you, then based the user's interest they can register for that. 
* It keep him notify about the schedule.
* It will help the users to find the useful events without wasting time searching around the events. Also, it ensures that the user wouldn't miss any events due to lack of time in searching the meetups.
* Also, based on the event agenda, it will help the user to prepare\\know some topics before attending the meeting and akong with framing some useful and interactive questions that user can ask in the webinar.





Daily Reporter

\-------------------

* It suggest the tasks that should be take action on, from our project, certification and events. Along with that, the user can add some more tasks of their preference and update the progress as happened.
* This is will help in validating our own work against the progress of our targets.
* It gives overall summary off our work impact and give insights to make our work much more better.
